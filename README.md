# parent-english-miniapp

这是一个用于学习和制作微信小程序的项目。

项目目标：做一个方便家长使用的英语启蒙工具，让英语一般的妈妈也能每天陪孩子说一句简单、实用的亲子英语。

## 一、产品定位

适合 0-8 岁孩子家庭使用的亲子英语启蒙小程序。

核心理念：

妈妈不用会很多英语，打开小程序就能照着说、照着做、照着陪孩子练。

## 二、目标用户

- 英语基础一般的妈妈
- 想给孩子做英语启蒙的家长
- 不知道每天和孩子说什么英语的家长
- 想要简单、直接、能马上使用的亲子英语内容的家长

## 三、第一版功能计划

- 每日一句亲子英语
- 按生活场景分类
- 英文句子 + 中文意思
- 音频跟读
- 妈妈动作提示
- 孩子回应示范
- 收藏常用句
- 每日打卡

## 四、第一版场景分类

- 起床
- 穿衣
- 洗漱
- 吃饭
- 出门
- 坐车
- 公园
- 玩具
- 洗澡
- 睡觉

## 五、第一版内容目标

先做 100 句亲子英语。

结构：

10 个生活场景 × 每个场景 10 句 = 100 句。

## 六、后续计划

第一阶段：整理内容表  
第二阶段：设计页面草图  
第三阶段：寻找小程序模板  
第四阶段：用微信开发者工具打开模板  
第五阶段：改成自己的英语启蒙小程序  
第六阶段：测试和优化
